jsJOMlib__check_date_string

function jsJOMlib__check_date_string(date,
format,
separator)

Check DATE against formats dd/mm/yyyy , d/mm/yyyy , dd/m/yyyy , d/m/yyyy.  Also allowed day and month inverse (american) position.  Default separator between day, month and year is “/”, but can be custom.

Parameters

datestring date to parse/check
formatstring containing one of the allowed date format: dd/mm/yyyy or mm/dd/yyyy
separator(optional) day-month-year inbetween character.  Must be the same as the format

Depends on

  • <jsSPESlib_math__is_number> function in jsSPESlib_math.js library file

Returns

  • false if any check is not passed.  Some debug can be sent to console
  • true if all checks are passed *
